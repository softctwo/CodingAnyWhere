## 🎯 Learning Paths by Goal

### **"I want to understand architecture from the ground up"**
→ Follow the core curriculum 1-6 in order
→ Estimated total time: 6-8 hours across multiple sessions

### **"I need to analyze an existing system"**
→ Start with [Multi-layered System Analysis](04-multi-layered-analysis.md)
→ Use [Architecture Diagrams](architecture-diagrams.md) for visualization
→ Reference [Glossary](architecture-glossary.md) for terminology
→ Estimated time: 2-3 hours

### **"I want hands-on practice"**
→ Quick read of [Project Architecture Basics](01-project-architecture-basics.md) (30 min)
→ Dive into [Practical Exercises](05-practical-exercises.md) (2-3 hours)
→ Use [Quick Reference](architecture-glossary.md) as needed

### **"I'm making architecture decisions right now"**
→ [Quick Reference & Glossary](architecture-glossary.md) - Decision frameworks
→ [Architectural Styles](03-architectural-styles.md) - Pattern selection
→ [Module Integration](02-module-integration.md) - Communication patterns
→ Estimated time: 30-60 minutes for specific decisions
